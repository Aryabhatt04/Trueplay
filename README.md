# 🎬 Flutter Video Player Pro

A **high-performance, gesture-driven video player** built with Flutter, inspired by modern media apps like MX Player and VLC — but designed from scratch with custom UX and advanced controls.

---

## 🚀 Features

### 🎮 Advanced Gesture Controls

* ⏩ Horizontal swipe → seek video
* 🔊 Vertical swipe (right) → volume control
* 💡 Vertical swipe (left) → brightness control
* ⚡ Long press → temporary 2x speed boost

### 🎥 Playback Experience

* ▶️ Play / Pause controls
* ⏩ Double tap → skip ±10 seconds
* 🎚 Adjustable playback speed (0.25x – 4x)
* 🔁 Resume playback from last position
* 🔒 Screen lock mode (prevents accidental touches)

### 🔍 Visual Enhancements

* 🔎 Pinch-to-zoom support
* 🎬 Frame preview while seeking
* 📸 Screenshot capture during playback

### 🔤 Subtitles

* 📂 Load external `.srt` files
* 🔄 Toggle subtitles ON/OFF
* ⚡ Auto-detect subtitles (same filename)

### 📁 Media Handling

* 📂 Browse and load local videos
* 📜 Watch history tracking
* 🧠 Smart playback memory

---

## 🧠 Tech Stack

* **Flutter (Dart)**
* `video_player` (ExoPlayer backend on Android)
* Native integrations:

  * Volume control
  * Screen brightness
* File handling & persistence
